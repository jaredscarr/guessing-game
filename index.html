Three Questions Assignment

About this page

  1. The JavaScript is prompting the user for their name and storing it inside a variable.
  2. It then displays a message to the user using their name.
  3. Then it asks the first question and saves the result into the variable question.
  4. It will check the users answers against the stored answers using a function.
  5. If the users ansers are correct it will store the amount of correct answers in another variable then tell the user how many they scored correctly at the end.
  6. It will repeat this process for the other two questions.
